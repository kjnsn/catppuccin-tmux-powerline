## Usage

See <https://github.com/erikw/tmux-powerline> for instructions on how to install tmux-powerline.

Copied from <https://github.com/erikw/tmux-powerline?tab=readme-ov-file#custom-theme>:

```bash
mkdir -p ~/.config/tmux-powerline/themes
curl https://raw.githubusercontent.com/catppuccin/tmux-powerline/refs/heads/main/catppuccin_<flavor>_theme.sh ~/.config/tmux-powerline/themes/catppuccin.sh
$EDITOR ~/.config/tmux-powerline/themes/catppuccin.sh
```

Remember to update the configuration file to use the new theme by setting `TMUX_POWERLINE_THEME=catppuccin`.
